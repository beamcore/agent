# Changelog

All notable changes to Beamcore will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [Unreleased]

### Added
- Chat-first TUI with scrollback, paste handling, and `@` file search.
- Provider-neutral chat through `Beamcore.Provider.Router`.
- Eeva runtime for model-executed Elixir under OTP supervision.
- Trusted-local path handling for relative, absolute, and symlinked paths.
- Supervised memory, ledger, provider scheduler, and task runtime.
- Alignment pause/resume for user corrections mid-turn.
- Token usage tracking with provider-reported or estimated counts.
- Theme system with 13 built-in themes.
- Multi-node mesh networking for distributed clusters.
- Release builds for Linux and macOS (x86 and ARM).
